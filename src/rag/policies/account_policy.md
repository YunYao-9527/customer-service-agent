# 账户管理政策

## 账户冻结

### 冻结条件

账户可能因以下原因被冻结：

1. **安全风险**
   - 检测到异常登录行为
   - 多次密码错误
   - 账户被盗风险

2. **违规行为**
   - 恶意刷单
   - 虚假交易
   - 欺诈行为

3. **司法要求**
   - 法院冻结令
   - 公安机关要求

4. **用户申请**
   - 用户主动申请冻结

### 冻结影响

- 账户余额不可使用
- 无法进行新的交易
- 无法提现
- 订单处理暂停

### 解冻条件

1. **安全风险解冻**
   - 验证用户身份
   - 确认账户安全
   - 修改密码

2. **违规行为解冻**
   - 提供申诉材料
   - 审核通过后解冻

3. **司法冻结解冻**
   - 提供解冻文件
   - 法院出具解冻裁定

## 身份验证

### 验证场景

以下操作需要身份验证：

- 修改绑定手机/邮箱
- 修改支付密码
- 大额提现（超过 1000 元）
- 账户冻结/解冻
- 退款到新账户

### 验证方式

1. **手机验证码**
2. **身份证验证**
3. **人脸识别**
4. **银行卡验证**

## 账户安全

### 安全建议

- 定期修改密码
- 开启双重验证
- 不在公共设备登录
- 及时绑定手机和邮箱

### 异常处理

发现账户异常时：
1. 立即修改密码
2. 联系客服冻结账户
3. 检查交易记录
4. 必要时报警处理
